**Create The Future of Music, Together.**

[audius.community](https://audius.community) is a glossary of community projects built on the [Open Audio Protocol](https://openaudio.org), using Audius APIs.

## Adding Your Project

To add your project and get free hosting or a redirect, submit an issue on [GitHub](https://github.com/AudiusProject/audius.community/issues/new?template=audius-community-listing.yml).

## Get Started with AI Agents

Send this to your AI agent to get started:

```
> Read https://audius.co/agents.md and follow the instructions to build with the Audius developer docs.
```

## Development

This project uses [`@audius/sdk`](https://www.npmjs.com/package/@audius/sdk) with React, TypeScript, and Vite. Add your `apiKey` and `apiSecret` to `.env` (see `.env.example` for reference).

```bash
npm run dev
```

For more details, check out the [Audius developer docs](https://docs.audius.co).

This project also uses the Audius design system [Harmony](https://www.npmjs.com/package/@audius/harmony).
